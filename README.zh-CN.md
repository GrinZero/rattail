### 特性

- 🛠️ &nbsp; 提供日常开发中经常使用的实用工具
- 🛠️ &nbsp; 工具实现非常轻量
- 🛠️ &nbsp; 使用 ts 编写，提供完善的类型支持

### 安装

```shell
# npm
npm i rattail -S
# yarn
yarn add rattail
# pnpm
pnpm add rattail
```

### 使用

```ts
import { isString } from 'rattail'

console.log(isString('rattail'))
```
