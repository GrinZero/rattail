# isNumber

Determine whether the input value is a number

### Usage

```ts
import { isNumber } from 'rattail'

isNumber(123) // return true
isNumber('rattail') // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
