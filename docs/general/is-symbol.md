# isSymbol

Determine whether the input value is a symbol

### Usage

```ts
import { isSymbol } from 'rattail'

isSymbol(Symbol('test')) // return true
isSymbol('rattail') // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
