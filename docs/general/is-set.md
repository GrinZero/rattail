# isSet

Determine whether the input value is a Set

### Usage

```ts
import { isSet } from 'rattail'

isSet(new Set()) // return true
isSet({}) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
