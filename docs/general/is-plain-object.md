# isPlainObject

Determine whether the input value is a plain object

### Usage

```ts
import { isPlainObject } from 'rattail'

isPlainObject({}) // return true
isPlainObject([]) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
