# toRawType

Return the raw type of the input value

### Usage

```ts
import { toRawType } from 'rattail'

toRawType({}) // return 'Object'
toRawType([]) // return 'Array'
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type   |
| :------: |
| `string` |
