# isArray

Determine whether the input value is an array

### Usage

```ts
import { isArray } from 'rattail'

isArray([]) // return true
isArray({}) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
