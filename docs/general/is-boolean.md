# isBoolean

Determine whether the input value is a boolean

### Usage

```ts
import { isBoolean } from 'rattail'

isBoolean(true) // return true
isBoolean('rattail') // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
